# AI Data Analysis Agent 🤖

## Overview
An AI-powered data analysis web application that automatically analyses any CSV dataset 
and generates comprehensive insights using Google Gemini 2.5 Flash — the latest AI model.

## Live Demo
Upload any CSV → AI analyses it instantly → Get business insights in seconds!

## Features
- 📊 Auto load and preview any CSV dataset
- 🔍 AI-powered comprehensive data analysis
- 📈 Interactive visualisations (Scatter, Bar, Line, Histogram)
- 💬 Natural language Q&A about your data
- 🎯 Identifies patterns, anomalies and business insights
- ⚡ Powered by Google Gemini 2.5 Flash

## Tech Stack
- Python
- Streamlit — web app framework
- Google Gemini 2.5 Flash — AI model
- Pandas — data processing
- Plotly — interactive charts

## How to Run
1. Clone the repository
2. Install dependencies: pip install streamlit google-generativeai pandas plotly
3. Add your Gemini API key in app.py
4. Run: streamlit run app.py

## Example Results
Tested on:
- Credit Risk Dataset — 150,000 records, 12 features
- Customer Churn Dataset — 7,043 records, 31 features
